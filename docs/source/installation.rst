Installation
============

Install from PyPI (recommended)
-------------------------------

.. code-block:: bash

    pip install phenosyn

---

Install from GitHub
-------------------

To install the development version directly from GitHub, first create a virtual environment:

.. code-block:: bash

    python3 -m venv venv
    source venv/bin/activate
    pip install --upgrade pip

Then clone the repository and install in editable mode:

.. code-block:: bash

    git clone https://github.com/P2GX/phenosyn.git
    cd phenosyn
    pip install -e .

---

Requirements
------------

- Python >= 3.10
